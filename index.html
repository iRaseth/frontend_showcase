Technologies :

I'm using :

In case of HTML, I think that I've got strong basics, and knowledge about HTML5. I guess that everyone starts adventure called "web-design" with learning basics of HTML and so I've done.

CSS is the brother of HTML, meanwhile HTML takes care of the correct layout, CSS does all what he can to create as attractive outlook as possible. Also in this case including CSS3 for sure, I have strong basic knowledge. I'm emphasizing the word "basic" because I don't really know how far I can go with my current knowledge.

In case of JavaScript, when I'm creating websites I do not overdose it. Sometimes when I'm doing some easy games that's the only way for me to accomplish it. I really want to learn more and more about it, due to the fact that the usage of JS is rapidly growing.

I'm used to Git Bash because from time to time I've been creating some projects in case of web-development. I think it's "must-have" nowadays as you can clearly see the changes throughout the process of creating certain part of the project.

I think that I can use Bootstrap without any problems, of course taking into consideration my "level". I'm well aware of the opportunities it opens with usage of CSS preprocessors, therefore I can't wait to learn it.

Superficial knowledge :

jQuery... JavaScript library, indeed in spite of lack of knowledge and practice with it, I can accelerate process of creating scripts. I think that it's great because it requires just basic knowledge of JS to just speed up your work. Nevertheless I'm certain that usage of jQuery on the higher level requires a lot of practice.
